# media-cleaner-site

Public website for the **Media Cleaner** Android app.

The only purpose of this repo is to host a stable, public Privacy Policy URL
required by the Google Play Store and AdMob.

## Live URLs

- Landing page: <https://gkna.github.io/media-cleaner-site/>
- Privacy policy: <https://gkna.github.io/media-cleaner-site/privacy-policy.html>

## Updating

Edit `privacy-policy.html`, bump the "Last updated" date at the top of the
file, commit, and push. GitHub Pages republishes within a minute.

```bash
git add privacy-policy.html
git commit -m "Update privacy policy"
git push
```
